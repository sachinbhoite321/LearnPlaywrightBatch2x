# Learn Playwright Batch 2x

Welcome to the **Learn Playwright Batch 2x** repository!  
This project contains hands-on JavaScript fundamentals and learning materials organized chapter-wise, aimed at building a strong foundation for Playwright test automation.

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/sachinbhoite321/LearnPlaywrightBatch2x.git
   cd LearnPlaywrightBatch2x
   ```

2. **Run JavaScript files with Node.js**
   ```bash
   node chapter_01_Basics/01_Basic.js
   ```

3. **Explore the chapters** in your favorite code editor (we recommend [VS Code](https://code.visualstudio.com/))!
